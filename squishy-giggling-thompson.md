# План создания сайта goQode

## Анализ презентации

### Ключевые сообщения бренда

**Позиционирование:**
- goQode — digital-агентство полного цикла
- Не продаёт услуги по отдельности, а проектирует системы
- Цифровой партнёр бизнеса, не подрядчик
- Средний чек ~6 000€

**УТП:**
- Интеграция: стратегия + UX/UI + разработка + маркетинг + аналитика
- Веб-разработка и маркетинг работают синхронно
- Предложения по задачам клиента, не по услугам

### 7 решений агентства

| # | Решение | Задача | Результат |
|---|---------|--------|-----------|
| 01 | Запуск бизнеса | Вывод на рынок с цифровой базой | Готовая точка продаж |
| 02 | Рост продаж | Повышение конверсии | Рост заявок и выручки |
| 03 | Онлайн-платформы | SaaS, маркетплейсы, кабинеты | Цифровой продукт |
| 04 | Event-решения | Цифровая инфраструктура событий | Управляемый event-продукт |
| 05 | Автоматизация | CRM, интеграции, процессы | Экономия ресурсов |
| 06 | Брендинг | Айдентика и позиционирование | Целостный бренд |
| 07 | Поддержка | Развитие после запуска | Стабильная работа продукта |

### Процесс работы (9 этапов)
1. Аналитика → 2. Стратегия → 3. UX-проектирование → 4. UI-дизайн → 5. Разработка → 6. Интеграции → 7. Тестирование → 8. Запуск → 9. Поддержка и рост

### Технологический стек
- **Frontend:** React, Next.js
- **Backend:** Node.js, Express, Fastify, NestJS
- **Mobile:** Flutter
- **CMS:** Webflow, WordPress
- **Databases:** PostgreSQL, Supabase, Firebase
- **Integrations:** API, Payments, CRM

## Видение сайта

### Концепция
Современный маркетинговый сайт digital-агентства с фокусом на:
- Визуальную идентичность (тёмная тема, lime-green акцент)
- Чёткую структуру по решениям
- Конверсию посетителей в заявки
- Премиальное восприятие (анимации, эффекты)

### Структура страниц

```
/                       # Главная
/about                  # О компании
/solutions              # Обзор всех решений
  /solutions/launch     # Запуск бизнеса
  /solutions/growth     # Рост продаж
  /solutions/platforms  # Онлайн-платформы
  /solutions/events     # Event-решения
  /solutions/automation # Автоматизация
  /solutions/branding   # Брендинг
  /solutions/support    # Поддержка
/cases                  # Портфолио (когда появятся кейсы)
/contact                # Контакты и форма заявки
```

## Дизайн-система

### Цвета
- **Background:** #0A0A0A (почти чёрный)
- **Foreground:** #FFFFFF (белый текст)
- **Accent:** #C9FD48 (lime-green)
- **Muted:** rgba(255,255,255,0.5)
- **Card:** rgba(255,255,255,0.05)
- **Border:** rgba(255,255,255,0.1)

### Типографика
- Заголовки: Bold, крупные (48-72px)
- Текст: Regular, 16-20px
- Шрифт: Inter или похожий sans-serif

### Анимации
- DecryptedText — эффект расшифровки для заголовков
- Fade + Blur при появлении секций
- Stagger-анимации для списков
- Three.js световой эффект на фоне (опционально)

### UI-компоненты
- Карточки с hover-эффектами
- Badges для категорий
- Numbered lists (01, 02, 03...)
- CTA-кнопки с gradient hover

## Технический стек для разработки

- **Framework:** Next.js 14+ (App Router)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **3D эффекты:** Three.js (опционально)
- **UI компоненты:** shadcn/ui
- **Формы:** React Hook Form + Zod
- **Email:** Resend или SendGrid
- **Деплой:** Vercel

## Этапы разработки

### Фаза 1: Базовая структура
- Настройка проекта Next.js
- Дизайн-система (цвета, типографика, компоненты)
- Layout с Header и Footer
- Главная страница

### Фаза 2: Страницы решений
- Шаблон страницы решения
- Генерация 7 страниц из данных
- Навигация между решениями

### Фаза 3: Контакты и конверсия
- Страница контактов с формой
- Интеграция отправки email
- CTA-секции на всех страницах

### Фаза 4: Полировка
- Анимации и микроанимации
- SEO (meta, og-tags, sitemap)
- Мобильная адаптация
- Оптимизация производительности

### Фаза 5: Портфолио (когда появятся кейсы)
- Структура данных для кейсов
- Страница списка кейсов
- Детальные страницы кейсов
- Связь кейсов с решениями
